How to use HTML

Boilerplate

You first have to place some basic tags. Every HTML project looks like this at the start:

First, you define the file as a HTML file, with the line

<!DOCTYPE HTML>

It then wraps the rest of the file in a wrapper saying that it is HTML.

<html>
...
</html>

Then, it defines the head of the document. This contains things such as the page title, which is set to be nothing. This is done by the lines

<head>
    <title></title>
</head>

Finally, it defines the body of the document to be empty. The body is where you put the actual content of the page.

<body>
    
</body>

The final blank page is like this.

<!DOCTYPE HTML>
<html>
    <head>
        <title></title>
    </head>
    <body>
        <!-- The webpage is empty -->
    </body>
</html>
